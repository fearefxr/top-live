## Token核心概念
Token是GPT处理文本的基本单位：
- 1000 Token ≈ 750英文单词或400-500汉字
- 不同模型Token上限：
  - GPT-3.5：约4096 Token
  - GPT-4：超30000 Token

使用官方Tokenizer工具可精确计算文本Token消耗。
